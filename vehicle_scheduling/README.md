# Vehicle Scheduling

## Description
This project solves the vehicle maintenance scheduling problem using the Knapsack Algorithm.

## Approach
- Each task has Duration (hours) and Impact (value)
- Each depot has limited Mechanic Hours (capacity)
- Selected tasks such that total duration ≤ capacity
- Maximized total impact

## Tech Used
- Node.js
- Axios (for API calls)
